# Tarefas 

### Cadastro de sensores: 
Implementação do sistema para registro e identificação de novos dispositivos de leitura no banco de dados.

### Monitoramento de temperatura:
 Coleta e exibição em tempo real dos dados térmicos do ambiente.

### Monitoramento de umidade do solo:
 Verificação constante do nível de hidratação da terra para suporte à decisão de rega.

### Controle automático de irrigação:
 Algoritmo que aciona o sistema de água baseando-se em parâmetros pré-definidos.

### Sistema de login:
 Criação da interface de entrada para garantir que apenas usuários autorizados acessem o sistema.

### Autenticação de usuários:
 Validação de credenciais e gerenciamento de sessões seguras.

### Registro de dados dos sensores: 
Armazenamento persistente de todas as leituras capturadas para consultas futuras.

### Histórico de umidade: 
Organização cronológica das variações de umidade para análise de padrões.

### Alertas de temperatura alta: 
Sistema de aviso imediato caso o calor ultrapasse o limite de segurança.

### Ajuste manual de iluminação: 
Interface para que o operador ligue ou desligue as luzes independente do modo automático.

### Integração com sensores IoT: 
Configuração da comunicação entre o hardware físico e o software via protocolos de rede.

### Dashboard de monitoramento: 
Painel visual centralizado com os principais indicadores e status do sistema.

### Previsão de irrigação:
 Estimativa de quando a próxima rega será necessária com base no consumo atual.

### Integração com previsão do tempo:
 Uso de dados meteorológicos externos para antecipar ajustes na estufa.

### Exportação de dados:
 Ferramenta para baixar os registros em formatos como CSV ou PDF.

### Detecção de falhas nos sensores:
 Rotina de verificação para identificar dispositivos offline ou com leituras inconsistentes.
